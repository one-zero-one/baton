# tillless-baton

A rubygem for coordinating Tillless interactions with scripted infrastructure like AWS and Heroku

## Installation

Add this line to your application's Gemfile:

gem 'tillless-baton'

And then execute:

$ bundle

Or install it yourself as:

$ gem install tillless-baton
